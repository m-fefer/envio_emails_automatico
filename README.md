Este projeto automatizou o processo de envio de e-mails personalizados com anexos em um ambiente profissional, reduzindo uma tarefa manual que levava mais de 15 horas para cerca de 10 minutos. 

🔧 O Que Ele Faz

- Compacta automaticamente pastas específicas contendo arquivos para diferentes clientes
- Associa cada cliente aos seus respectivos e-mails com base em uma tabela no Excel
- Personaliza o assunto e o corpo de cada e-mail
- Fornece feedback no console sobre o status de envio de cada e-mail

⚠️ Observação

Todas as informações sensíveis, como endereços de e-mail e credenciais, foram removidas ou ocultadas por motivos de privacidade.
